# AyunBlog-Vue3

基于 Vue 3 + TypeScript 的轻量级博客系统。

## 1. 项目概述

### 1.1 项目背景

本项目旨在开发一个简单、直观的博客系统，博主和普通用户可以直接通过前端界面编辑、发布文章，访客仅能浏览已发布的文章。系统不包含后台管理功能，所有操作均通过前端完成。

### 1.2 技术栈

- **前端框架**: Vue 3 (Composition API)
- **编程语言**: TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **UI 组件库**: Element Plus
- **网络请求**: Axios
- **编辑器**: md-editor-v3 (Markdown)

## 2. 功能需求

### 2.1 用户管理

- **角色划分**:
  - **博主 (Admin)**: 拥有最高权限，可管理所有文章、分类、标签。
  - **普通用户 (User)**: 可发布、编辑和删除自己的文章，不能操作他人文章。
  - **访客 (Guest)**: 仅能浏览已发布的文章。
- **登录注册**: 基于邮箱的注册与登录系统。

### 2.2 博客内容管理

- **文章发布**: 支持 Markdown 编辑，支持草稿保存、定时发布。
- **文章展示**: 首页文章列表按时间倒序排列，支持按分类、标签、日期筛选。
- **分类与标签**: 博主可创建/编辑/删除分类和标签；用户仅能选择。

### 2.3 社交与 SEO

- **社交分享**: 支持分享到微博、微信（扫码）、Twitter 等平台。
- **SEO 优化**:
  - 动态更新页面 `title`。
  - 自动生成 `meta description` 和 `keywords`。
  - 支持 OpenGraph 标签，优化社交分享预览效果。

## 3. 页面设计与交互逻辑

- **简约大厂风**: 参考 Medium、知乎、GitHub Blog 等设计风格，保持界面简洁、专业。
- **响应式设计**: 完美适配 PC、平板和手机端。
- **沉浸式阅读**: 详情页采用大字号、高对比度及合理的行间距，提升阅读体验。
- **动效优化**: 使用平滑的过渡动画，提升用户交互感知。
- **深色模式**: 支持一键切换深浅色主题（可选）。

### 3.1 导航栏优化

- **整合入口**: 移除菜单栏中的“发布文章”文字链接，统一使用右侧的“写文章”操作按钮。
- **快捷操作**: “写文章”按钮旁增加下拉菜单，包含“发布文章”、“草稿箱”入口，符合主流创作平台（如掘金、知乎）的设计。
- **用户菜单**: 个人主页入口保留在用户头像下拉菜单中。

### 3.2 分类与标签页 (重构)

- **卡片式网格**: 采用非对称的卡片网格布局，分类卡片包含背景图或图标，增强视觉引导。
- **交互效果**: 悬浮时展示该分类下的文章数量和最近更新时间。
- **标签集成**: 标签云移至页面侧边或底部，按热度排序并支持搜索过滤。

### 3.3 搜索与筛选结果页

- **状态感知**: 页面顶部清晰展示当前筛选条件（如：“搜索：Vue3”、“分类：前端开发”）。
- **结果反馈**: 增加搜索结果数量提示及“未找到相关内容”的优雅缺省页。
- **排序切换**: 支持按“最新发布”、“最多阅读”等维度进行结果二次排序。

### 3.4 草稿箱

- **列表管理**: 采用卡片或列表形式，展示草稿标题、最后修改时间、所属分类。
- **快速操作**: 支持在列表中直接进行“编辑”、“删除”操作，无需进入详情页。

### 3.5 关于页面

- **多维展示**: 包含个人简介、技术栈、项目经历、联系方式等模块。
- **视觉风格**: 采用时间轴或故事卡片形式，增加页面的交互感和故事性。

## 4. 目录结构说明

- `src/layouts`: 页面通用布局
- `src/views`: 页面视图组件
- `src/components`: 通用 UI 组件
- `src/stores`: 状态管理
- `src/api`: 接口请求封装
- `src/utils`: 工具函数封装
- `src/styles`: 全局样式定义
